AutoHotkey is a free, open-source scripting language for Windows that allows users to easily create small to complex scripts for all kinds of tasks such as: form fillers, auto-clicking, macros, etc.

You can get AutoHotKey at https://autohotkey.com/ and it's a great tool to make your day to day and work life a ton easier!

You can take each of these individually or you can plug the data into your main AHK file and have it running with the rest of your scripts in a centeral location.
